**Fundamentos del internet y redes**

El internet puede ser mostrado como una red de redes mediante la cual se conectan distintos aparatos tecnológicos como los computadores o teléfonos móviles. El internet viaja a partir de routers los cuales crean rutas mediante las cuales se desplaza para llegar de un dispositivo a otro.



**Protocolos de Comunicación (TCP/IP, DNS y DHCP)**

las computadores tienen un lenguaje propio con el cual se comunica con protocolos **TCP/IP**.



**IP:** Asigna cada dispositivo una forma de identificación numérica y única para cada dispositivo para diferenciarlo. Tiene 2 formas de asignarlos las cuales son:



IPv4: usa 32 bits, permite a 4 mil millones de usuarios tener una dirección única, pero actualmente es insuficiente.



IPv6: Esta se usa actualmente gracias a que su base esta em 128 bits lo cual                         otorga una cantidad exagerada de direcciones para el futuro.



TCP: Es quien garantiza el envió de los datos de forma segura siendo el quien se encarga de en tal caso de una perdida de datos se reenvié de nuevo utiliza 2 puertos dependiendo del trafico de datos necesario como el 80 que se usa para el trafico web estándar y el 443 usado para un trafico mas privado.



DNS: es un sistema que simplifica los nombres que utiliza los computadores para facilitar la forma de recordarlo del ser humano.



DHCP: es quien facilita la conexión a una red recibiendo fácilmente una dirección IP sin tener que configurarla de forma manual.



**Protocolo HTTP**

Es el protocolo mediante el cual se envían y reciben paginas web.



**URL:** Es la dirección completa que mantiene un esquema como http o https junto a un dominio como Google acompañado de el TDL y una ruta de archivo solicitado.



Verbos HTTP: Son las acciones que el usuario pide y el servidor es capaza de responder utilizando GET para solicitar algún tipo de información y POST para enviarla.



Códigos de estado: Numero mediante el cual el servidor indica el resultado de una acción utilizando distintos códigos como 200 para mostrar un éxito, 301 para cuando el sitio web se allá movido a otra dirección y 404 cuando el archivo no es encontrado.
